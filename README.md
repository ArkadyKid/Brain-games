## About
Pack of some simple mathematics games.

## Requirements

node v.13+

## Publish
````
$ make publish
````

## Install
```
$ make install
$ npm link
````

## List of games

### Even number

#### About
Answer "yes" if the number is even, otherwise answer "no".

#### Run
````
$ brain-even
````
